# 采访计划 - 智能植物护理系统

## 采访要求

根据EE542课程要求：
- **最少采访人数**: 6人（团队成员数×2）
- **采访形式**: 当面、Zoom或电话（不能是问卷）
- **目标**: 验证项目价值，确认问题真实存在
- **录音**: 需录音并包含在最终演示视频中（需获得许可）

---

## 采访对象分类

### Category 1: 垂直农场/温室运营者（2人）

**目标人群**:
- 垂直农场经理
- 温室种植者
- 室内农业技术人员

**联系方式**:
- LinkedIn搜索: "vertical farm manager"
- 本地垂直农场公司
- 农业科技创业公司

**核心问题**:

1. **问题验证**
   - 你们目前如何管理灌溉和光照？
   - 使用固定时间表还是传感器驱动？
   - 每月水电费用大约是多少？

2. **痛点挖掘**
   - 植物死亡率大约是多少？主要原因是什么？
   - 是否遇到过浇水过多或过少的问题？
   - 需要多少人力监控植物状态？

3. **解决方案验证**
   - 如果有AI系统可以自动优化浇水和光照，你觉得有价值吗？
   - 你最关心的指标是什么？（节水、存活率、人力成本）
   - 你认为合理的价格是多少？（$10K/年 vs $50K/年）

4. **技术接受度**
   - 你们是否使用过自动化系统？效果如何？
   - 对AI/机器学习系统的信任度？
   - 最担心的问题是什么？（可靠性、成本、复杂度）

---

### Category 2: 智能家居产品经理/工程师（2人）

**目标人群**:
- 智能家居设备公司PM
- IoT设备工程师
- 智能种植器硬件开发者

**联系方式**:
- Click & Grow, AeroGarden等公司员工（LinkedIn）
- 智能家居创业公司
- USC校友在相关公司工作

**核心问题**:

1. **市场验证**
   - 你们的智能种植器使用什么控制算法？
   - 用户最常抱怨的问题是什么？
   - 植物存活率目标是多少？实际是多少？

2. **技术挑战**
   - 目前最大的技术难点是什么？
   - 为什么不使用机器学习优化？（成本、复杂度、算力）
   - 嵌入式设备的GPU/算力限制有多严重？

3. **商业模式**
   - 你们的客户最看重什么功能？
   - 硬件 vs 软件订阅收入占比？
   - 如果有更智能的算法，愿意支付多少授权费？

4. **集成可行性**
   - 你们的系统是否有API或SDK？
   - 对接第三方AI算法的难度？
   - 数据隐私和云端依赖的考量？

---

### Category 3: 家庭园艺爱好者（2人）

**目标人群**:
- 认真的家庭园艺爱好者
- 拥有多种植物的用户
- 曾经杀死过植物的人（几乎所有人）

**联系方式**:
- USC园艺俱乐部
- Reddit r/houseplants
- 朋友/同学推荐

**核心问题**:

1. **痛点验证**
   - 你有多少植物？有多少死掉了？
   - 最难的是什么？（浇水频率、光照、出差忘记照顾）
   - 每周花多少时间照顾植物？

2. **现有解决方案**
   - 你用什么方法提醒自己浇水？
   - 是否用过智能种植器？体验如何？
   - 手机App提醒vs自动浇水，你更喜欢哪个？

3. **产品需求**
   - 如果有完全自动的植物护理系统，你会买吗？
   - 可接受的价格范围？（$100, $200, $500）
   - 最看重的功能？（节水、高存活率、易用性、外观）

4. **情感因素**
   - 杀死植物时的感受？（有罪感、沮丧、不再尝试）
   - 自动化会减少养植物的乐趣吗？
   - 你希望系统完全自主还是可以手动干预？

## 采访流程模板

### 1. 开场（2分钟）

```
你好！我是USC EE542的学生Kelin Wu。

我正在开发一个AI驱动的智能植物护理系统，使用强化学习来优化浇水和光照。
这个项目旨在解决传统固定时间表浪费50%水资源和40%植物死亡的问题。

我想了解你在植物护理方面的经验和挑战，这将帮助我验证这个问题是否值得解决。

采访大约需要15-20分钟，我会录音用于课程项目演示，可以吗？
```

### 2. 核心问题（15分钟）

- 按上述分类问题提问
- 鼓励讲故事（"能举个具体例子吗？"）
- 追问细节（"那时你怎么想的？"）
- 观察情绪反应（是否真的在意这个问题？）

### 3. 方案演示（3分钟）

```
我的解决方案是这样的：

[展示系统架构图/演示视频]

- 使用传感器实时监测土壤、温度、光照
- AI智能体通过强化学习自主决定何时浇水、开灯
- 目标：节省40%水资源，提升植物存活率至85%+

你觉得这个方案有价值吗？有什么建议或担忧？
```

### 4. 收尾（2分钟）

```
非常感谢你的时间和见解！你的反馈对我很有帮助。

项目完成后，我会分享最终结果。如果感兴趣，可以提供你的邮箱。

再次确认：可以使用录音在课程演示中吗？（仅面向教授和同学）
```

## 注意事项

⚠️ **隐私保护**:
- 在录音前明确告知并获得许可
- 如果对方要求匿名，使用"Industry Expert"等称呼
- 不泄露商业机密信息

⚠️ **学术诚信**:
- 真实记录，不编造采访内容
- 准确引用，不断章取义
- 保存原始录音以备查证

⚠️ **时间管理**:
- 尽早开始联系（至少提前2周）
- 预留时间应对拒绝和改期
- Week 1-2完成所有采访
